# Learn Single Table Design with DynamoDB

A Go application demonstrating single table design patterns with DynamoDB.

## Requirements

- Go 1.24+
- Docker
- Docker Compose

## Commands

Run tests:

    make test

Code coverage:

    make test-coverage

Build:

    make build

Run:

    make run
  
We host a local dynamodb instance with an admin panel that can be accessedd at:

    http://locahost:8001
